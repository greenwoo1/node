# ControlNode (CN) - Infrastructure Management System

ControlNode - це повноцінна система управління інфраструктурою з нейронним дизайном та повним контролем над серверами, доменами, користувачами та фінансами.

## Технологічний стек

### Frontend:
- **HTML5, CSS3, JavaScript (ES6+)**
- Чистий JavaScript без фреймворків
- AJAX для взаємодії з API
- Адаптивний дизайн (Desktop, Tablet, Mobile)

### Backend:
- **FastAPI** (Python)
- **PostgreSQL** (база даних)
- **JWT** авторизація
- **SQLAlchemy** ORM
- **Alembic** міграції

### Інфраструктура:
- **Docker & Docker Compose**
- **Nginx** reverse proxy
- Мульти-контейнерна архітектура

## Швидкий старт

### Передумови:
- Docker & Docker Compose
- Git
- 4GB+ RAM
- Linux/MacOS/Windows (з WSL2)
